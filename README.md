# Lightning Trade - 量化交易系统

## 项目介绍
Lightning Trade 是一个基于 Spring Boot 的量化交易系统，集成了老虎证券 API，支持美股和港股的自动化交易。系统提供了回测功能，可以在历史数据上测试交易策略的表现。

### 主要特性
- 支持美股和港股市场
- 提供多种交易策略（MA、MACD、布林带等）
- 完整的回测系统
- 实时交易执行
- 历史数据管理
- 交易记录追踪

## 项目架构

### 核心模块
1. **策略模块** (`strategy`)
   - 实现了多种交易策略
   - 支持策略的动态配置和切换

2. **回测模块** (`backtest`)
   - 支持在历史数据上测试策略
   - 提供详细的回测报告
   - 支持不同市场的交易成本计算

3. **数据服务** (`service`)
   - 历史数据获取和管理
   - 实时行情数据处理
   - 数据持久化

4. **交易执行** (`execution`)
   - 订单管理
   - 交易执行
   - 仓位管理
